# 业界特快


[**欧盟资助开源软件**](https://github.com/ruanyf/weekly/blob/master/docs/issue-40.md)
* 很多开源项目都靠志愿者的奉献，并没有经费来源
* 现在，为了提高开源软件的安全性，欧盟宣布为14个开源项目的漏洞提供赏金计划
* 如果你发现了这些软件的漏洞，欧盟官方会向你支付奖金。每个项目的奖金总额在3万～9万欧元。
* 这14个开源项目如下
   * 7-zip，Apache Kafka，Apache Tomcat，Digital Signature Services (DSS)，Drupal，Filezilla， FLUX TL，the GNU C Library (glibc)，KeePass，midPoint，Notepad++，PuTTY，the Symfony PHP framework，VLC Media Player，WSO2
